2020-01-03

CVE _2019-1-3

#web服务器单个网站

一、发布网站

1、安装Apache服务

# yum install -y httpd

#systemctl start httpd.service

/etc/httpd/conf：该目录存放Apache服务器的配置文件

/var/www/html：该目录是Apache服务器的默认站点根目录

/etc/httpd/logs:Apache服务器的日志文件目录

2、测试Apache是否安装成功

在浏览器的地址栏输入：localhost进行测试

3、移动到目录var/www/html/下

将公司网站exam复制到/var/www/html/目录下，并修改配置文件：

Vim /etc/httpd/conf/httpd.conf

​ 修改：DocumentRoot “/var/www/html/“为

​ DocumentRoot “/var/www/html/exam”

【修改完配置文件后必须要重启服务：systemctl restart httpd】

二、发布个性化网站

项目考核信息网站部署完成后，考虑到站点是内部网站，为了保证安全，将首页改为exam.html，并使用8080端口访问。

改首页名字：

#mv /var/www/html/exam/index.html /var/www/html/exam/exam.html

修改配置文件：

Vim /etc/httpd/conf/httpd.conf

​ Listen 8080

​ DirectoryIndex exam.html index.html

重启服务：

systemctl restart httpd

访问测试：192.168.75.128:8080

Web服务器虚拟主机

管理员需要通过基于域名或IP地址的虚拟主机方式布署多个网站。

项目实施：

1、基于域名的虚拟主机

（1）域名注册，在/etc/hosts中追加两行：

192.168.75.128 exam.excesoft.com

192.168.75.128 attendance.excesoft.com

域名测试：nslookup

（2）将两个网站放到/var/www目录下

（3）修改主配置文件：vim /etc/httpd/conf/httpd.conf

​ Listen 80

​ DirectoryIndex index.html

在末尾增加：Include conf/vhost/*.conf

（4）在/etc/httpd/conf 目录下新建vhost目录，并在下面新建exam.conf与attendance.conf两个

域名配置文件，并作如下编辑。

#vim exam.conf

<VirtualHost 192.168.75.128>

ServerName exam.excesoft.com 域名

DocumentRoot /var/www/exam/ 站点

<Directory “/var/www/exam/”>

Options Indexes FollowSymLinks //显示网页文件列表

AllowOverride NOne //忽略.htaccess 文件

Require all granted //允许所有

#vim attendance.conf

<VirtualHost 192.168.75.128>

ServerName attendance.excesoft.com

DocumentRoot /var/www/attendance/

<Directory “/var/www/attendance/”>

Options Indexes FollowSymLinks //显示网页文件列表

AllowOverride NOne //忽略.htaccess 文件

Require all granted //允许所有

(5) 使用httpd -t检查虚拟主机的配置是否正确

(6) 重启Apache服务并进行测试

2、基于IP地址的虚拟主机

（1）设置多个IP地址

（2）域名注册

（3）修改主配置文件：vim /etc/httpd/conf/httpd.conf

Listen 80

DirectoryIndex index.html

在末尾增加：Include conf/vhost/*.conf

（4）为每个主机新建配置文件并定义一个容器

（5）重启Apache服务器并进行测试
